# Pipeline-CI-CD-Tratamento-de-dados-com-ML
Atividade somativa de DevOps. Criação de um pipeline CI/CD com tratamento de dados (ETL) em Python e utilização de Machine Learning (regressão) com a biblioteca Scikit-Learn para criação de uma tabela Excel com previsão de vendas (de loja fictícia) para o ano de 2026.
